# Prompt bom 1

## Prompt usado

Papel: Você é um instrutor de programação e revisor de código. 

Contexto: Estou criando um app simples de tarefas para estudar IA Coding. 

Stack: HTML, CSS e JavaScript puro. 

Objetivo: Implementar apenas a estrutura inicial do projeto. 

Restrições:
- Não usar frameworks.
- Não usar backend.
- Não usar bibliotecas externas.
- Não gerar tudo de uma vez.
- Antes de codar, explicar o plano.

Critérios de aceite:
- Criar index.html, styles.css e script.js.
- Explicar a função de cada arquivo.
- O app deve abrir direto no navegador.
- O código deve ser simples para iniciante.

Formato da resposta:
1. Plano
2. Estrutura de arquivos
3. Código
4. Como testar
5. Possíveis erros

## Objetivo
Gerar estrutura inicial do app.

## O que funcionou
- A IA respeitou a stack.
- Separou plano e código.
- Explicou os arquivos.
- Incluiu forma de teste.

## O que ainda pode melhorar
- Pedir comentários no código.
- Pedir checklist de revisão.
